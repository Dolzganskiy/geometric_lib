# geometric_lib

## circle.py

### def area(r)
- Название: area
- Назначение: вычисляет площадь круга по данному радиусу
- Подпись area(r)
- Параметры: r - радиус круга типа float
- Возвращаемое значение: площадь круга в виде числа типа float
- Формула: area(r) = πR²
- Пример работы: area(3) = 28,27433388
### def perimetr(r)
- Название: perimetr
- Назначение: вычисляет периметр круга по данному радиусу
- Подпись perimetr(r)
- Параметры: r - радиус круга типа float
- Возвращаемое значение: периметр круга в виде числа типа float
- Формула: area(r) = 2πR
- Пример работы: area(3) = 18,84955592

## rectangle.py

### def area(a, b)
- Название: area
- Назначение: вычисляет площадь прямоугольника по данным длинам его сторон
- Подпись area(a, b)
- Параметры: a, b - длины сторон прямоугольника в виде чисел типа float
- Возвращаемое значение: площадь прямоугольника в виде числа типа float
- Формула: area(a, b) = ab
- Пример работы: area(3, 7) = 21,0
### def perimetr(a, b)
- Название: perimetr(a, b)
- Назначение: вычисляет периметр прямоугольника по данным длинам его сторон
- Подпись perimetr(a, b)
- Параметры: a, b - длины сторон прямоугольника в виде чисел типа float
- Возвращаемое значение: периметр прямоугольника в виде числа типа float
- Формула: perimetr(a, b) = 2a + 2b
- Пример работы: perimetr(3, 7) = 20,0
